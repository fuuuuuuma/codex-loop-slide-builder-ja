# DESIGN.md — インディゴ・カーム（Indigo Calm）

> **これは何**：スライドの「配色・フォント・型」をひとまとめにした正典です。`make-slides` の `DESIGN.md` としてコピーするか、
> エージェントに「この DESIGN.md に従ってスライドを作って」と渡すだけで、この見た目で統一されたデッキになります。
> 番号 05 / 50 ／ カテゴリ: ビジネス ／ モード: ライト（明背景）

## 使いどころ
- 研究・分析レポート
- 雰囲気：静か・上質
- モチーフ：インディゴの番号バッジ

## カラートークン（意味ベース）
| トークン | 役割 | 値 |
|---|---|---|
| `--brand` | 主色（見出し・強調） | `#4338CA` |
| `--brand-deep` | 主色の濃い版（小見出し・帯） | `#312E81` |
| `--accent` | 差し色（番号・枠線・バッジ・矢印） | `#818CF8` |
| `--accent-deep` | 差し色の濃い版（バッジ影など） | `#4F46E5` |
| `--bg` / `--paper` | 背景 / カード白面 | `#F6F6FF` / `#FFFFFF` |
| `--ink` / `--ink-mid` | 本文 / 補助テキスト | `#312E81` / `#54527F` |
| `--ink-sub` | 極淡（**罫線など非テキスト専用**） | `#9A98C4` |
| `--on-accent` | accent 地の上に乗せる文字色 | `#312E81` |

## そのまま貼り付け（`:root`）
スライドHTMLの `:root{ ... }` をこのブロックで丸ごと置き換えてください。

```css
:root{
  --brand:#4338CA; --brand-deep:#312E81; --brand-soft:rgba(67,56,202,0.08); --brand-line:rgba(67,56,202,0.22);
  --accent:#818CF8; --accent-deep:#4F46E5; --accent-soft:rgba(129,140,248,0.16);
  --bg:#F6F6FF; --paper:#FFFFFF; --ink:#312E81; --ink-mid:#54527F; --ink-sub:#9A98C4;
  --line:rgba(49,46,129,0.12); --on-accent:#312E81; --canvas:#DEDDF4;
  --head:'Noto Sans JP';
}
```

> フォント（見出し）：**Noto Sans JP**。Google Fonts を `<head>` に追加：
> `<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;700;900&display=swap" rel="stylesheet">`
> 本文は Noto Sans JP 推奨。

## スライドの型（このテーマで使う基本レイアウト）
1. 表紙（大見出し＋一言サブ）
2. 結論（TL;DR・3カード）
3. 悩み（箇条書き）
4. 〇〇とは（定義・比喩）
5. 全体像（フロー図 ① → ② → ③ → ④）
6. 3本柱（カード×3）
7. before / after（2パネル比較）
8. 比較表（見出しは ink 地＋白字）
9. 使い方（悩み → こう頼む）
10. まとめ＋CTA・出典

## コントラスト規約（必守）
- `--accent`／`--accent-deep` は **枠線・地色・バッジ専用**。**小さい文字色に使わない**。
- 文字色は `--ink` / `--ink-mid` / `--brand-deep`（背景に対して AA を確保）。
- バッジ・ボタン・番号は **accent 地 ＋ `--on-accent` の文字**（このテーマでは `#312E81`）。
- `--ink-sub` は罫線・区切りなど**非テキスト専用**。本文・キャプションには使わない。
- 比較表の見出しは **ink 地＋白文字**。

## 禁止表現（誇張・断定保証）
`必ず / 絶対 / 確実 / 裏技 / 神 / ヤバい / 衝撃 / 100%稼げる` は使わない。効果は中立に（「〜も目指せる」）。
