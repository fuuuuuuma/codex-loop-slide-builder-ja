# DESIGN.md — ダーク・テック（Dark Tech）

> **これは何**：スライドの「配色・フォント・型」をひとまとめにした正典です。`make-slides` の `DESIGN.md` としてコピーするか、
> エージェントに「この DESIGN.md に従ってスライドを作って」と渡すだけで、この見た目で統一されたデッキになります。
> 番号 27 / 50 ／ カテゴリ: ダーク ／ モード: ダーク（暗背景）

## 使いどころ
- 開発・AI・テック実演
- 雰囲気：近未来・締まり
- モチーフ：ネオンのライン（暗背景）

## カラートークン（意味ベース）
| トークン | 役割 | 値 |
|---|---|---|
| `--brand` | 主色（見出し・強調） | `#22D3EE` |
| `--brand-deep` | 主色の濃い版（小見出し・帯） | `#67E8F9` |
| `--accent` | 差し色（番号・枠線・バッジ・矢印） | `#22D3EE` |
| `--accent-deep` | 差し色の濃い版（バッジ影など） | `#06B6D4` |
| `--bg` / `--paper` | 背景 / カード白面 | `#0B1220` / `#131C2E` |
| `--ink` / `--ink-mid` | 本文 / 補助テキスト | `#E6EDF7` / `#9FB2CC` |
| `--ink-sub` | 極淡（**罫線など非テキスト専用**） | `#5E768C` |
| `--on-accent` | accent 地の上に乗せる文字色 | `#0B1220` |

## そのまま貼り付け（`:root`）
スライドHTMLの `:root{ ... }` をこのブロックで丸ごと置き換えてください。

```css
:root{
  --brand:#22D3EE; --brand-deep:#67E8F9; --brand-soft:rgba(34,211,238,0.14); --brand-line:rgba(34,211,238,0.3);
  --accent:#22D3EE; --accent-deep:#06B6D4; --accent-soft:rgba(34,211,238,0.16);
  --bg:#0B1220; --paper:#131C2E; --ink:#E6EDF7; --ink-mid:#9FB2CC; --ink-sub:#5E768C;
  --line:rgba(255,255,255,.10); --on-accent:#0B1220; --canvas:#060A12;
  --head:'Noto Sans JP';
}
```

> フォント（見出し）：**Noto Sans JP**。Google Fonts を `<head>` に追加：
> `<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;700;900&display=swap" rel="stylesheet">`
> 本文は Noto Sans JP 推奨。

## スライドの型（このテーマで使う基本レイアウト）
1. 表紙（大見出し＋一言サブ）
2. 結論（TL;DR・3カード）
3. 悩み（箇条書き）
4. 〇〇とは（定義・比喩）
5. 全体像（フロー図 ① → ② → ③ → ④）
6. 3本柱（カード×3）
7. before / after（2パネル比較）
8. 比較表（見出しは ink 地＋白字）
9. 使い方（悩み → こう頼む）
10. まとめ＋CTA・出典

## コントラスト規約（必守）
- `--accent`／`--accent-deep` は **枠線・地色・バッジ専用**。**小さい文字色に使わない**。
- 文字色は `--ink` / `--ink-mid` / `--brand-deep`（背景に対して AA を確保）。
- バッジ・ボタン・番号は **accent 地 ＋ `--on-accent` の文字**（このテーマでは `#0B1220`）。
- `--ink-sub` は罫線・区切りなど**非テキスト専用**。本文・キャプションには使わない。
- 比較表の見出しは **ink 地＋白文字**。

## 禁止表現（誇張・断定保証）
`必ず / 絶対 / 確実 / 裏技 / 神 / ヤバい / 衝撃 / 100%稼げる` は使わない。効果は中立に（「〜も目指せる」）。
