# DESIGN.md 50選 — スライド配色・型の正典カタログ

受け取った人が **いつでも選んで使える** スライドデザイン50種です。各ファイルは独立した正典で、
`make-slides/DESIGN.md` にコピーするか、エージェントに「この DESIGN.md に従って」と渡すだけで、その見た目に統一されます。

- 使い方：好きなテーマの `.DESIGN.md` を開く → 中の `:root{ ... }` ブロックをスライドHTMLに貼る → 完成。
- すべて **コントラスト規約・禁止表現** を内蔵済み（そのまま本番に使えます）。
